# PowerBI-Sales-Insights
This project showcases an interactive Power BI dashboard designed to analyze sales performance, customer behavior, and product demand across different regions.

The dashboard provides meaningful insights through data modeling, DAX calculations, and visual storytelling.

- Key Insights
Top 10 countries based on revenue
Top 10 customers contributing to revenue
Regional demand analysis for products
Time-series analysis of sales trends
Identification of high-performing products

- Tools & Technologies Used
Data Cleaning & Transformation
Data Modeling
Data Visualization
DAX (Data Analysis Expressions)
